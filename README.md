# Follow-Me Robot Project
**Milestone 1: Perception and Environment Setup**

## Overview
A ROS 2 Jazzy robot that identifies a blue cylinder using OpenCV and tracks it in Gazebo Harmonic.

## Setup
- **ROS 2**: Jazzy Jalisco
- **Simulator**: Gazebo Harmonic
